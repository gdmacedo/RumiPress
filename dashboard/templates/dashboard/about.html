Sobre este projeto

Este projeto foi realizado no tempo de aproximadamente em 5 hgoras, onde foi desenvolvido um aplicativo web de controle de despesas para a empresa de distribuição de livros: Rumi Press.

Neste projeto, foi utilizado as habilidades em Python e Django para construir modelos de dados, criando estruturas de aplicativos usando Django Admin e criamos relatórios visuais em aplicativos da web.

Ao final deste projeto, foi produzido um aplicativo web que apoiará a equipe da Rumi Press no gerenciamento de despesas de distribuição de livros.

Também publicamos este projeto no Github, conforme segue o endereço abaixo, compartilhanso as descobertas com potenciais empregadores com a criação uma página de portfólio do Coursera.

Este projeto foi projetado para permitir que fosse usado várias abordagens para criar uma solução.

Essa flexibilidade tem a missão de que, com que esse projeto seja um destaque da concorrência e ajuda na apresentação das prática no ambito empregador a diferenciar sas habilidades das de outros.
